📊 Datasets
MFJS01–MFJS10: Modified flexible job shop instances
Mk01–Mk10: Benchmark instances adapted with maintenance constraints

Each .fjs file defines:

Number of jobs and machines
Operation sequences
Alternative machines and processing times


📎 Code Availability

This repository contains a partial implementation of the proposed methods.

The full source code is currently not publicly available because the corresponding paper is under peer review. To ensure a fair and anonymous review process, we will release the complete implementation after the paper is accepted.
